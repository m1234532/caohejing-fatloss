# 漕河泾减脂餐厅推荐 Web App

一个基于漕河泾印象城的减脂友好餐厅推荐应用，可安装到手机桌面使用。

## 功能

- 🍽️ 正餐快餐餐厅列表（60家）
- 🥤 饮料甜品餐厅列表（26家）
- ⭐ 按星级筛选（5星、4星、3星）
- 🔍 搜索餐厅或菜系
- 📱 可安装到手机桌面
- 💾 支持离线使用

## 如何使用

### 方法一：直接打开
直接在浏览器中打开 `index.html` 文件

### 方法二：部署到免费托管（推荐）

#### Vercel 部署
1. 将整个文件夹上传到 GitHub 仓库
2. 在 [vercel.com](https://vercel.com) 导入仓库
3. 点击 Deploy

#### GitHub Pages 部署
1. 将整个文件夹上传到 GitHub 仓库
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支和 `/ (root)` 文件夹
4. 等待部署完成

### 安装到手机桌面

#### iOS (Safari)
1. 用 Safari 打开网页
2. 点击底部的分享按钮
3. 向下滚动，点击"添加到主屏幕"

#### Android (Chrome)
1. 用 Chrome 打开网页
2. 点击右上角的菜单按钮
3. 点击"安装应用"或"添加到主屏幕"

## 技术栈

- Vue 3
- Vant UI
- PWA (Progressive Web App)

## 数据来源

数据来自 `data.js` 文件，可以直接编辑该文件来更新餐厅数据。
